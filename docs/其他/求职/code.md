# 手撕代码

## 目标

意向职位：高性能计算（HPC）和 AI 基础设施（AI Infra）方向。

需要准备的内容：

- 机试：基础数据结构和算法就行
- 面试：可能拷打并行编程、CUDA 等

## 思维

- 问题澄清：确认输入输出边界（如“空输入如何处理？”）
- 举例验证：设计常规/边缘用例（如超大矩阵、全零输入）
- 暴力解法：先给出基础方案（如 O(n²) 解法）
- 优化分析：识别重复计算/无效操作（如 DP 状态复用）
- 代码实现：模块化编写 + 即时验证（边写边测临界条件）
