    1. mac_addr:

    2. 是许多结构（如 lacpdu, bond_marker, aggregator, port, port_params, ad_system）中用来表示 MAC 地址的基础单元。

    3. lacpdu 和 bond_marker:

    4. 这两个结构描述了链路聚合控制协议（LACP）和标记协议（Marker Protocol）的数据包结构，用于链路聚合的通信。

    5. aggregator:

    6. 表示链路聚合的一部分，包含 MAC 地址、聚合相关的键值，以及关联的 port 和 slave 指针。

    7. 每个 aggregator 可以包含多个 port。

    8. port:

    9. 描述链路聚合中的一个端口，包含 actor 和 partner 的信息以及状态机的状态。
    10. 包含指向其所属的 slave 和 aggregator 的指针。

    11. 与 lacpdu 直接相关，因为其发送和接收的 LACP 信息存储在 lacpdu 字段中。

    12. port_params:

    13. 用于存储端口的系统、优先级、键值等信息。

    14. port 中有两个 port_params（partner_admin 和 partner_oper），分别代表管理和操作的伙伴信息。

    15. ad_system:

    16. 表示链路聚合系统中全局的系统优先级和系统 MAC 地址。

    17. 被 ad_bond_info 使用。

    18. bond_3ad_stats:

    19. 用于统计 LACP 和 Marker PDU 的发送、接收和错误信息。

    20. 被 ad_bond_info 和 ad_slave_info 使用。

    21. ad_bond_info 和 ad_slave_info:

    22. 这些结构分别存储链路聚合的全局信息和特定从属设备（slave）的信息。
    23. ad_slave_info 包含一个 aggregator 和一个 port。

    master 注册阶段¶

    • bond_init(struct net_device *bond_dev)：注册为 bond_netdev_ops.ndo_init，调用链如下

      ret = dev->netdev_ops->ndo_init(dev);
    register_netdevice(struct net_device *dev) net/core/dev.c include/linux/netdevice.h
        bond_create() drivers/net/bonding/bond_main.c include/net/bonding.h
            bonding_init()

      关于内核网络设备的注册过程，见 Documentation/networking/netdevices.rst。

    • bond_create(struct net *net, const char *name)：负责创建一个新的 bonding 设备

      struct net_device *bond_dev = alloc_netdev_mq(..., bond_setup, ...);
struct bonding *bond = netdev_priv(bond_dev);
dev_net_set(bond_dev, net);
register_netdevice(bond_dev);
netif_carrier_off(bond_dev);
bond_work_init_all(bond);

    • bond_setup(struct net_device *bond_dev)：初始化 bonding 网络设备的各种属性和默认参数

      struct bonding *bond = netdev_priv(bond_dev);
bond->params = bonding_defaults;
bond->dev = bond_dev;
ether_setup(bond_dev);
bond_dev->netdev_ops = &bond_netdev_ops;
bond_dev->flags |= IFF_MASTER;
...

    • bond_work_init_all(struct bonding *bond)：完成延迟工作任务的初始化

      INIT_DELAYED_WORK(&bond->ad_work, bond_3ad_state_machine_handler);
INIT_DELAYED_WORK(&bond->slave_arr_work, bond_slave_arr_handler);

      详见 Deferred work。

    slave 注册阶段¶

    slave 及其初始化

    static struct slave *bond_alloc_slave(struct bonding *bond,
          struct net_device *slave_dev)
{
 struct slave *slave = NULL;

 slave = kzalloc(sizeof(*slave), GFP_KERNEL);
 if (!slave)
  return NULL;

 slave->bond = bond;
 slave->dev = slave_dev;
 INIT_DELAYED_WORK(&slave->notify_work, bond_netdev_notify_work);

 if (bond_kobj_init(slave))
  return NULL;

 if (BOND_MODE(bond) == BOND_MODE_8023AD) {
  SLAVE_AD_INFO(slave) = kzalloc(sizeof(struct ad_slave_info),
            GFP_KERNEL);
  if (!SLAVE_AD_INFO(slave)) {
   kobject_put(&slave->kobj);
   return NULL;
  }
 }

 return slave;
}
