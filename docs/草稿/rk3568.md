## 刷写系统

### OpenWRT

刷 OpenWRT 的教程不在 RK3568 文档，而在 Linux 开发指南：[1. OpenWRT 手册 — Firefly Wiki](https://wiki.t-firefly.com/zh_CN/Firefly-Linux-Guide/manual_openwrt.html)。

刷入 EMMC 的步骤如下：

- 进入 Maskrom 模式，烧录 NorFlash2eMMC 固件
- 上一步自动重启进入 Loader 模式，修改 Image 位置从 0x0 开始烧录 OpenWRT Img 镜像到 eMMC
- 上一步自动重启，进入 OpenWRT 系统

该 OpenWRT 的源码位于 [Firefly-rk-linux-utils/lede](https://github.com/Firefly-rk-linux-utils/lede)
